# FitMind Gym Routine

검색 없이 바로 운동하려고 만든 개인 루틴 웹앱입니다.

## 실행 방법
1. `c:\MyApp\FitMind AI\index.html` 파일을 브라우저로 열기
2. 요일 선택 후 운동 진행
3. `세트 완료` 버튼으로 세트 누적
4. 휴식 타이머 확인 후 다음 세트 진행
5. 운동 종료 후 `오늘 요약 저장`

## 포함 기능
- 루틴 순서 자동 진행 (MON~FRI)
- 운동 가이드 카드
  - 운동방법
  - 기구 설명
  - 공(짐볼) 대체 운동
  - 주의 포인트 / 자주 하는 실수
- 세트 완료 및 휴식 타이머
- 완주율/검색 횟수/불안도 기록
- 최근 기록 저장(LocalStorage)
